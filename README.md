# Festival Urbano 2026

## Descripción del proyecto

Este proyecto consiste en el diseño y desarrollo de una página web informativa para un evento ficticio llamado “Festival Urbano 2026”.

El objetivo es presentar de forma clara y atractiva un festival de música urbana incluyendo información sobre el evento, artistas y venta de entradas, utilizando Bootstrap 5 y aplicando control de versiones mediante GitHub.

---

## Estructura de la página

La web está compuesta por las siguientes secciones obligatorias:

- Navbar responsive con menú de navegación.
- Hero principal con título, descripción y botón de acción.
- Sección informativa con sistema grid (row + col-md-6).
- Sección de artistas destacados usando grid (row + col-md-4).
- Footer con información final.

Sistema de columnas utilizado:
- col-md-6 en la sección informativa (2 columnas).
- col-md-4 en la sección de artistas (3 columnas).

---

## Componentes Bootstrap utilizados

- Navbar
- Grid system (container, row, col)
- Cards
- Botones (btn, btn-primary)
- Clases de espaciado (p-5, my-5, mb-3)
- Clases de tipografía (display-3, lead, fw-bold)

## Control de versiones (commits realizados)

1. Estructura base HTML y conexión Bootstrap  
   Se creó el archivo index.html y se añadió la conexión mediante CDN a Bootstrap 5.

2. Añadida navbar y hero principal  
   Se añadió la barra de navegación y la sección principal destacada del evento.

3. Añadidas secciones con grid y footer  
   Se completó la estructura con secciones organizadas en columnas y el footer final.

4. Mejora diseño final del festival urbano  
   Se mejoró el diseño general añadiendo componentes como cards, mejor organización visual y estructura más completa.

---

## Mayor dificultad encontrada

La mayor dificultad fue comprender el funcionamiento del sistema de columnas de Bootstrap y la gestión de ramas en GitHub.

La solución fue estructurar el trabajo en diferentes commits organizados y utilizar correctamente el sistema row + col para lograr que la página fuera responsive en distintos dispositivos.
