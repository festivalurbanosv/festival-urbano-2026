# Festival Urbano 2026

## Descripción del proyecto

Este proyecto consiste en el diseño de una página web informativa para un evento juvenil relacionado con la músic.

La idea del evento es crear un espacio donde los jóvenes puedan disfrutar de conciertos en directo y también aprender sobre el proceso creativo de hacer música. Durante el evento habría conciertos, charlas de artistas y talleres donde se explicaría cómo se escriben canciones y cómo se producen temas musicales.

La web está hecha utilizando Bootstrap 5 para que el diseño sea responsive y se adapte a móvil, tablet y ordenador.

---

## Estructura de la página

La página está organizada en varias secciones principales para mostrar la información del evento de forma clara.

- Navbar: barra de navegación superior donde aparecen las diferentes secciones de la página: Inicio, Artistas, Entradas y Contacto.

- Inicio: sección principal de la web donde se presenta el evento "SoundLab Joven 2026", con una breve descripción, la fecha y el lugar donde se celebra.

- Artistas: sección donde se muestran algunos de los artistas que participan en el evento y que realizarán conciertos o charlas sobre su proceso creativo.

- Entradas: sección donde se informa sobre la asistencia al evento y cómo conseguir las entradas.

- Contacto: sección final donde aparece información de contacto para resolver dudas o pedir más información sobre el evento.

- Footer: pie de página sencillo con el nombre del evento.

---

## Columnas utilizadas

En el diseño se ha utilizado el sistema grid de Bootstrap con las siguientes columnas:

- col-md-6 para la sección de información (2 columnas).
- col-md-4 para la sección de actividades (3 columnas).

En este proyecto se han utilizado algunos componentes básicos de Bootstrap:
